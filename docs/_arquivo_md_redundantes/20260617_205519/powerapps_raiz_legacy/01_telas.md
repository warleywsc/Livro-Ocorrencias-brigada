## Regra de nomenclatura

- Prefixo `scr` para telas.
- Prefixo `cmp` para componentes.
- Prefixo `gal` para galerias.
- Prefixo `frm` para formulários.
- Prefixo `btn` para botões.
- Prefixo `lbl` para textos.
- Prefixo `var` para variáveis.
- Prefixo `col` para coleções.

## Observação

Os nomes técnicos podem ser curtos, mas os rótulos exibidos ao usuário devem permanecer em português do Brasil.
