# Validacao com Dados Ficticios

Projeto: Livro de Ocorrencias Digital da Brigada de Incendio
Data: 09/06/2026
Status: Validacao manual principal realizada

## 1. Site validado

https://etnweb.sharepoint.com/sites/BrigadaIncendio

## 2. Listas envolvidas

- Usuarios
- Plantoes
- EquipePlantao
- LocaisOperacionais
- PendenciasOperacionais
- Ocorrencias

## 3. Dados ficticios cadastrados

- Locais operacionais ficticios cadastrados.
- Plantao ficticio criado.
- Pendencia ficticia criada.
- Ocorrencia ficticia criada.

## 4. Validacoes realizadas

- A lista LocaisOperacionais foi corrigida para usar URL propria.
- Foi identificado e corrigido problema de coluna Observacoes duplicada em LocaisOperacionais.
- O plantao ficticio foi criado e exibido corretamente em Plantoes.
- A pendencia ficticia foi vinculada ao plantao e ao local operacional.
- A ocorrencia ficticia foi vinculada ao plantao, ao local operacional e a pendencia gerada.
- Campos de consulta exigiram selecao do item sugerido pelo SharePoint, nao apenas texto digitado.
- Campo PendenciaGerada reconheceu corretamente o item quando selecionado como etiqueta.

## 5. Observacoes importantes

- Os dados usados foram ficticios e servem apenas para validacao estrutural.
- Campos do tipo Pessoa podem exigir usuario institucional real.
- Campos do tipo Consulta devem ser selecionados a partir da sugestao exibida pelo SharePoint.
- Colagem em modo de grade pode falhar quando houver campos de Pessoa, Consulta ou muitas colunas.

## 6. Resultado

A validacao estrutural principal foi considerada aprovada.
